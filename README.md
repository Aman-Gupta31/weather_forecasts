### Weather Forecasts
## Overview
* The Weather Dashboard is a web application that allows users to check the current weather and a 5-day forecast for a specific city. It utilizes the OpenWeatherMap API to retrieve weather data and provides a user-friendly interface for a seamless experience.

## Features
* Current Weather Display: Shows the current temperature, humidity, and wind speed     for a selected city.
* 5-Day Forecast: Displays a forecast for the next 5 days, including temperature, humidity, and wind speed.
* Search History: Keeps track of recently searched cities, making it easy for users to revisit past searches.
 
## Getting Started
Clone the repository to your local machine.
* git clone https://github.com/Aman-Gupta31/Weather-Dashboard-master
 
## Navigate to the project directory:
* cd weather-dashboard
* Open the index.html file in a web browser.


## Usage
* Enter the name of a city in the search bar.
* Click the search button or press Enter to retrieve weather information.
* View the current weather, 5-day forecast for the selected city.
* Check the search history for quick access to past searches.
